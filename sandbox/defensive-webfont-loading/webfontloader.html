Webfont Loading Demo

This page uses the webfont loader to bring in fonts from Fontdeck. It uses a normal font stack of webfont followed by fallback fonts in the font-family property. This makes the webfonts work once they've loaded.

In order to display text whilst the fonts are loading, we've added in duplicate styles preceded by a .wf-loading class on each selector which omitting the webfont from the font-family.
